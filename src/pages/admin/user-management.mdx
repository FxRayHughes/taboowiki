## 🔧 API 接口

### 获取所有用户
```bash
GET /admin-api/users
Authorization: Bearer <token>
```

### 获取用户详情
```bash
GET /admin-api/users/{id}
Authorization: Bearer <token>
```

### 更新用户状态
```bash
PUT /admin-api/users/{id}/status
Authorization: Bearer <token>
Content-Type: application/json

{
  "status": 1
}
```

### 设置管理员权限
```bash
PUT /admin-api/users/{id}/admin
Authorization: Bearer <token>
Content-Type: application/json

{
  "isAdmin": 1
}
```
